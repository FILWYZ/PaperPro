# PaperPro Scholar Assistant

PaperPro Scholar Assistant 是一款面向科研人员、研究生和工程技术人员的 Chrome 浏览器插件，  
用于 **快速检索学术论文、生成 GB/T 7714-2015 标准引用，并一键跳转学术镜像站搜索**。

---

## ✨ 核心功能

### 🔍 多数据库统一检索
- 基于 **Semantic Scholar Graph API**
- 覆盖：
  - arXiv
  - IEEE
  - Springer
  - ACM
  - PubMed 等主流学术来源
- 无需登录、无需 API Key

---

### 📚 GB/T 7714-2015 引用自动生成
- 自动格式化作者（≤3 人 / 超出显示“等”）
- 自动识别文献类型：
  - `[J]` 期刊论文
  - `[C]` 会议论文
- 支持 arXiv 预印本标注
- 一键复制引用文本，适用于：
  - 论文写作
  - 学位论文
  - 科研报告

---

### 🔎 学术镜像站跳转
- 内置默认 Google Scholar
- 支持 **自定义镜像站地址**
- 当 API 无结果或结果不理想时：
  - 一键跳转镜像站继续检索

---

### 🧠 智能兜底机制
- API 无结果 → 自动提示镜像搜索
- API 请求失败 → 仍提供人工检索入口
- 避免“空白页面”体验

---

## 🧩 使用场景

- 文献调研与快速引用生成
- 国内网络环境下的学术检索
- 不依赖 EndNote / Zotero 的轻量级解决方案
- 作为写作前的“第一步筛选工具”

---

## ⚙️ 插件设置说明

### 自定义镜像地址
1. 点击插件设置按钮
2. 输入镜像站地址（自动补全 `https://`）
3. 保存后立即生效

示例：https://scholar.google.com

---

## 📐 GB/T 7714 说明

本插件实现的是 **工程实用版 GB/T 7714-2015 引用生成规则**，特点如下：

- 优先保证：
  - 作者格式正确
  - 文献类型标注明确
  - 引用结构规范
- 对以下内容做了简化处理：
  - 卷期页码缺失情况
  - 预印本类型归类差异

如需期刊投稿级别的严格格式，请以期刊官方模板为准。
